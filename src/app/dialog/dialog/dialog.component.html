Stock App

About - Why this app?
  • This is a simple Stock App dashboard.
  • Developed and hosted for personal use.
  • This app doesn't track stock price in real time.
  • Don't provide any personal or sensitive data to this app.
 What this app does?
  • This app displays a dashboard for Stocks listed in US Stock Exchange using ticker symbol.
  • This app requires stock ticker symbol, number of stocks and average price it was purchased for.
  • The INFO icon on the tool bar displays the sample dashboard which doesn't require login
How to use?
  • To access the app first enter User Name and Password then click Sign Up button.
  • Once logged in, use Add Stock button to add stock symbol, number of stock and average price.
  • Once added the dashboard will start providing the metrics.
